Spatial Decision Support System

2,500 Acres in the Adirondacks

In 2025, New York voters passed a constitutional amendment requiring the state to acquire 2,500 acres of land "equal to or greater in value" to offset land taken from the Forest Preserve. The amendment never defined value — so I built a system that does, and that recommends which parcels to buy under different stakeholder priorities.

Role
Design, modeling, code
Tools
ArcGIS Pro, Python (arcpy), Leaflet
Method
Multi-criteria evaluation
Course
GTECH 732, Hunter College

The problem

The amendment retroactively authorized ski and biathlon trails built on 323 acres of protected Adirondack Forest Preserve, and in exchange requires New York to add 2,500 new acres of comparable ecological value. But "value" is subjective: a conservation group, the state land manager, and a taxpayer will each rank candidate parcels differently. The tool makes those priorities explicit and shows how the recommended acquisition changes as they shift.

Explore the recommendations

Pick a stakeholder profile to see which parcels the model recommends. Click any parcel for details.

Each result is the highest-ranked parcel — or, when no single parcel fits the target acreage, a contiguous cluster the model grows by absorbing the best-scoring neighbors while keeping the shape compact. All ten profiles run against the same underlying parcel scores; only the weights differ.

How it works

The model runs in two halves. A one-time preprocessing stage scores every 30-metre cell of the Adirondacks on four ecological factors — land cover & slope suitability, riparian proximity, rare-habitat value (EDGE), and protection-gap value — and summarizes each factor onto candidate parcels as a per-acre mean. Then, for a given stakeholder, those factor scores plus parcel size and cost are normalized and combined into a single weighted index; the highest-scoring parcel or grown cluster wins.

Keeping the ecological scores as per-acre means (rather than totals) is what lets the six criteria stay independent — a big tract and a small tract of equal habitat quality score the same, and only the explicit size weight rewards bigness.

What the results show

The ten profiles above trace the whole trade-off. A fiscal profile, minimizing cost under a tight budget, settles on a single affordable parcel in Hopkinton; a biodiversity profile assembles a compact, higher-value cluster of rare habitat in Parishville; and the water-weighted profiles shift the recommendation to Long Lake, near the rivers and lakes. Because the model enforces each profile's spending cap as a hard limit and only grows a contiguous cluster when a single parcel is too small, every result stays both affordable and manageable as one connected acquisition.

Grounding in the literature

The normalize-then-weight structure follows Li & Nigh (2011); the continuous slope-suitability curve follows Store & Kangas (2001); riparian buffers follow Wenger (1999) and the Adirondack Park Agency shoreline rules; rare-habitat ranks are cardinalized with Rank-Order-Centroid weights (Barron & Barrett 1996); and the compact-cluster growth borrows the boundary-length logic of Marxan (Watts et al. 2009).
